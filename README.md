# GEstimator

GEstimator is a simple civil estimation software written in Python and GTK+. GEstimator can prepare estimates along with rate analysis and supports multiple databases.

## Installation

Binary files can be downloaded from the website https://manuiisc.blogspot.in/p/blog-page_28.html

## Dependencies:

### Python 3 (v3.4)

Python Modules:

* appdirs (v1.4.3)
* et_xmlfile
* jdcal
* openpyxl (v2.4.9)
* peewee (v2.10.2)
